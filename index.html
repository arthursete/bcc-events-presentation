01 Introduction

▼

Context

Brisbane City Council publishes comprehensive venue data through their Open Data Portal, but the information is presented in a raw, technical format inaccessible to everyday users. BCC Events Finder proposes to transform this data into an intuitive venue discovery platform.

02 User Story

▼

2.1 Application Description

BCC Events Finder — a web application allowing Brisbane residents and visitors to search, filter, and explore 1,696 council event venues by suburb, venue type, or keyword.

ComponentTechnology
FrontendHTML, CSS, JavaScript
BackendPython (Flask)
DatabaseSQLite
MappingLeaflet.js

2.2 Existing Solution — BCC Open Data Portal

The current portal (data.brisbane.qld.gov.au) is a data catalogue built on Opendatasoft. It provides table view, map view, and export/API access but is designed for developers, not the general public.

Key Cons: Not designed for general public, no suburb-based browsing, raw data presentation, no venue type visual filtering, no mobile optimisation, overwhelming technical interface.

3.2 Data Dictionary — 5 Normalised Tables

Suburbs (SuburbID PK, SuburbName) — 186 records

VenueTypes (TypeID PK, TypeName) — 10 records

Venues (VenueID PK, VenueName, VenueAddress, Lat, Long, SuburbID FK) — 1,696 records

VenueTypeLink (VenueID FK, TypeID FK) — junction table for many-to-many

Users (UserID PK, Username, PasswordHash, Role, CreatedDate) — admin auth

4.1 HCI — Design Principles

PrincipleApplication
ContrastDark nav bar vs light content; blue primary vs red delete buttons
AlignmentLeft-aligned form labels; consistent table columns
RepetitionConsistent venue card design across all pages
ProximityRelated filters grouped in sidebar; venue info + map side by side
Visual HierarchyLarge hero heading; venue name as largest text on detail page

4.1 HCI — Usability Features

  • Consistent navigation on all pages
  • Prominent search bar in hero section
  • Breadcrumb navigation on detail pages
  • Results count and sort options
  • Pagination for 1,696 records
  • Related venues for further discovery
  • Accessible: clear labels, high contrast, required field markers
